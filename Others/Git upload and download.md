
·Git upload
```
1. Initialize: git init
2. Add files: git add .
3. Check status: git status -s
4. Commit: git commit -m "<comment>"
5. Add source: git remote add origin <address> / git remote add another <address>
6. Upload: git push origin master
7. Upload to another repo: git push --mirror <address>
```

·Git download
```
1. Cancel local change: git stash
2. Download: git pull origin
```
